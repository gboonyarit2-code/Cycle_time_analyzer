Flexible Cycle Time & Buffer Analyzer

วิเคราะห์ Cycle Time ของไลน์การผลิตแบบกำหนดโครงสร้างเองได้: จำนวนเครื่องรวม, Process, Machine รายเครื่อง, Single/Parallel, ประเภทเครื่องจักร, Loader และตำแหน่ง Buffer ที่ควรจัดวาง

Machine + Loader + Buffer รองรับการสร้างไลน์ผลิตหลาย Process พร้อม Animation และ Buffer Map

1) ตั้งค่าโครงสร้างไลน์และเกณฑ์วิเคราะห์

วิธีคำนวณที่ใช้ใน Web App นี้
  • Variance % = (Actual CT - Standard CT) / Standard CT × 100
  • Process แบบ Parallel คำนวณ Effective CT = 1 / ผลรวม(1 / Actual CT ของเครื่องใน Process นั้น)
  • Line Actual CT = ค่าสูงสุดระหว่าง Actual Effective CT ของแต่ละ Process และเวลา Loader ตามโหมดที่เลือก
  • Line EFF. เทียบ Standard = Standard Line CT / Actual Line CT × 100
  • Output/hr = 60 / Actual Line CT เพราะ CT ในโปรแกรมนี้เป็นหน่วยนาทีต่อชิ้น
  • Buffer Score พิจารณาจากตำแหน่งก่อน/หลังคอขวด, Downstream CT ช้ากว่า Upstream, จุดหน้า Parallel, จุดรวมจาก Parallel เข้า Single และ Loader Transfer ที่ช้าหรือเกิน Std
  • จำนวน Buffer ขั้นต่ำ เป็น heuristic เริ่มต้นเพื่อวิเคราะห์ไลน์: High ≈ 10 นาทีของ Line CT, Medium ≈ 5 นาที, Low ≈ 2 นาที ควรปรับด้วยข้อมูล Downtime, Changeover, Minor stop และพื้นที่จริง
